# download 163Music

### 要求
需要[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)支持
#### 安装
```
git clone git@github.com:Binaryify/NeteaseCloudMusicApi.git
npm install
```
#### 运行
```
node app.js 
```
打来另一个终端运行
```python
python3 get_music.py
```
输入歌单的ID
例如：23655387
